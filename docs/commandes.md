# Liste des commandes à utiliser pour gérer le projet

## Linux (terminal)

### Se deplacer dans un dossier
```
cd <nom_du_fichier>
```
### Lister les elements dans le dossier courant
```
ls -al 
```
### Créer un `dossier` vide
```
mkdir <nom_du_dossier>
```
### Créer un `fichier`
```
touch <nom_du_fichier>
```
### Renommer un `fichier` / `dossier`
```
mv <nom_du_fichier/dossier>
```
### Deplacer un fichier / dossier
```
mv <nom_du_fichier/dossier> <destination>
```
